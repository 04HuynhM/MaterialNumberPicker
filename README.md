## Gradle Dependency

Add this in your root `build.gradle` file:

```gradle
allprojects {
	repositories {
		// ... other repositories
		maven { url "https://jitpack.io" }
	}
}
```

Then add the dependencies that you need in your project.

```gradle
dependencies {
  compile "com.github.StephenVinouze:MaterialNumberPicker:{latest_version}"
}
```
